# AI/ML LDT vs IVD-SaMD Classification Tool

**A free interactive teaching resource from [TechInHSR LLC](https://techinhsr.com)**

---

## What This Is

A three-slide interactive decision aid for IRB professionals, HRPP staff, regulatory compliance personnel, and laboratory directors who need to determine whether an AI/ML-enabled laboratory-developed test (LDT) functions as a regulated IVD software function (IVD-SaMD) under FDA's device framework.

The tool walks through a four-question working test, explains what changed and what held under the current administration, and provides concrete examples across the not-device / borderline / device-like spectrum.

---

## Why This Exists

FDA's 2024 final rule — which explicitly classified IVDs manufactured by laboratories as medical devices and announced a four-year enforcement phase-out — was vacated by a federal district court (E.D. Tex.) on March 31, 2025. FDA formally rescinded the rule and reverted 21 CFR 809.3(a) to its pre-2024 text effective September 19, 2025.

The result: most LDTs have reverted to a CLIA-centered oversight posture, and the broad device-classification framework no longer has regulatory force.

What did not change: FDA's authority over AI-enabled device software functions remains intact. FDA continues to regulate AI/ML-enabled medical devices, maintain its AI-Enabled Medical Device List, and issue guidance on software as a medical device — including 2025 draft guidance on AI device lifecycle management.

The line between a CLIA-governed laboratory service and an FDA-regulated IVD software function has never been harder to draw. This tool is designed to make that distinction teachable.

## Who Should Use This

- IRB members and HRPP staff reviewing AI/ML-enabled laboratory tests in research protocols
- Regulatory affairs and compliance professionals advising laboratories on FDA vs. CLIA jurisdiction
- Educators teaching SaMD classification, the CDS framework, or FDA device regulation
- Investigators and laboratory directors assessing whether an AI/ML tool requires FDA engagement

## How to Use It

No installation required.

1. Download `ai_ml_ldt_decision_tree.html`
2. Open it in any modern web browser
3. Navigate slides using arrow keys, swipe (mobile), or the on-screen controls

The tool runs entirely in the browser. No data is collected or transmitted.
